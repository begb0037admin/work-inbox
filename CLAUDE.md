## Conventions
- Single `index.html` — no framework, no build step
- `fetch_inbox.py` derives output path dynamically — safe to move the folder
- Dashboard uses `localStorage` (keys: `workInbox_briefings_v1`, `workInbox_today_v1`, `workInbox_ticks_v1`)
- `data/inbox_raw.json` is gitignored — never commit it (contains email content)
- Dashboard imports JSON via the Import panel (paste JSON → Load)
