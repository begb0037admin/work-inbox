## Standing Rules
- Never commit tokens or raw data
- All GitHub writes via Contents API (PAT from GITHUB_PAT env var)
- Always backup before any change — naming convention: fetch_inbox_backup_YYYY-MM-DD-description.py
- Seat A never references local disk — all reads via GitHub proxy or API
- Task Scheduler requires elevated PowerShell to Register/Unregister tasks
- Unicode em dashes in repo files silently break PowerShell .Replace() — use split-on-section-headers
- No em dashes in PowerShell commit messages — plain hyphens only
- Manual script execution currently broken — use schtasks /run only
- DO NOT run python fetch_inbox.py directly from PowerShell until COM session issue resolved
